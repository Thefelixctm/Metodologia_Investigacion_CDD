# App Streamlit - Unidad 3

**Tema:** Modelamiento predictivo explicable del riesgo de burnout académico en estudiantes universitarios mediante técnicas de Ciencia de Datos.

## Archivos incluidos

- `app.py`: aplicación principal de Streamlit.
- `requirements.txt`: dependencias mínimas.
- `data/articulos.csv`: matriz breve de evidencia bibliográfica usada por la app.

## Ejecución local

Desde la carpeta del proyecto:

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Contenido de la app

La aplicación comunica el planteamiento de investigación de la Unidad 3:

1. Portada.
2. Problema de investigación.
3. Estado del arte: fundamentos, avances y brechas.
4. Pregunta de investigación.
5. Hipótesis.
6. Objetivo general y objetivos específicos.
7. Evidencia bibliográfica.
8. Coherencia problema-brecha-pregunta-hipótesis-objetivos.
9. Declaración de uso de IA.
10. Checklist de cumplimiento de la rúbrica.

## Declaración de uso de IA

Se utilizó ChatGPT como apoyo para organizar el planteamiento, sintetizar evidencia, estructurar la app y mejorar la redacción. La selección de estudios, interpretación final y validación del contenido fueron revisadas manualmente por el estudiante.
