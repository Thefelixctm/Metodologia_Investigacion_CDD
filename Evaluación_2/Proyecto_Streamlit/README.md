# Presentación interactiva en Streamlit

Proyecto: **Predicción y modelado del burnout académico en estudiantes universitarios mediante técnicas de Ciencia de Datos**.

## Cómo ejecutar

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Estructura

- `app.py`: portada, problema, preguntas, área OCDE y ODS.
- `pages/1_Metodologia_PRISMA.py`: estrategia de búsqueda, criterios, PRISMA y síntesis.
- `pages/2_Estudios_incluidos.py`: artículos incluidos y related work.
- `pages/3_Resultados_visualizaciones.py`: resultados, variables, métodos y operacionalización.
- `pages/4_Discusion_vacios.py`: discusión, vacíos y proyección.
- `pages/5_ODS.py`: alineación con ODS 3, 4 y 9.
- `pages/6_IA_y_material_suplementario.py`: uso de IA, verificación humana y descargas.

## Requisitos cubiertos

La aplicación incluye título, integrante, problema de investigación, preguntas, área OCDE y ODS, estrategia de búsqueda, diagrama PRISMA, artículos incluidos, resultados principales, visualizaciones interactivas, related work, metodologías de Ciencia de Datos, discusión, vacíos de investigación, declaración de uso de IA, referencias principales y material suplementario descargable.
